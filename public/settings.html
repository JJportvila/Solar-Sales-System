Google Drive 备份配置

开启谷歌备份后，请填写 Folder ID 和 Access Token，并先测试连通。

Google Drive Folder ID
在目标文件夹地址中，`folders/` 后面的内容就是 Folder ID。
打开 Google Drive
Google Drive Access Token
常用 scope：
https://www.googleapis.com/auth/drive.file
https://www.googleapis.com/auth/drive
打开 Google OAuth Playground 查看 Drive 授权说明
一键复制获取 Folder ID 和 Access Token 的步骤
1. 打开 Google Drive 目标文件夹，复制 `folders/` 后面的 Folder ID。
2. 打开 Google OAuth 2.0 Playground。
3. 选择 `https://www.googleapis.com/auth/drive.file` 并授权。
4. 交换 token 后复制 Access Token，粘贴回这里。
5. 先保存，再测试连通。
